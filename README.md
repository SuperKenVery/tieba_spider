# OpenSpiders

#### 介绍
这是一个众多爬虫项目的代码合集，持续更新，欢迎start

#### 软件架构
本项目是基于python框架开发的，其内容包含百度贴吧爬虫、新浪微博爬虫、知乎爬虫、前程无忧爬虫等常见的爬虫项目
